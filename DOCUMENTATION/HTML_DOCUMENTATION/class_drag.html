Detailed Description

The Drag Class This is attached to all GameObjects that can be dragged and all GameObjects a dragged GameObject may possibly collide with. It handles all drag functionality, as well as "Win Conditions".

The OnTriggerEnter method. This is called whenever two blocks with rigid bodies collide. If they both possess the "isWinner" bool, and it is true, then "Master" GameObject is retrieved, and the level is changed.

The OnTriggerExit method. This is called whenever two blocks previously in a collision, resolve that collision. This removes the isLastSelected trait from the object and resets its blockid to the blockidcopy.

The OnTriggerStay method. This is called whenever two blocks with rigid bodies collide and remain in collision. This method corrects the collision by moving the collided block from whence it came. This was adapted from the more complicated bits of drag.js in previous development.
